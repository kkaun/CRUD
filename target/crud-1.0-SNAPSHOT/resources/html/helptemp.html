Помощь в работе со списком

  • Таблица-список с данными о пользователях заполняется посредством Ваших действий в Меню администратора.


  • Редактирование полей с пользовательскими данными осуществляется с помощью двойного щелчка по требуемому полю, после чего в поле вводятся новые данные.


  • Сортировать список можно по параметрам: "Имя", "Возраст", "Администратор?" , "Дата регистрации".
    Для восходящей или нисходящей сортировки по требуемому параметру достаточно нажать на графу с параметром: 1 раз - для восходящей сортировки; 2 раза - для нисходящей сортировки; 3 раза - для возвращения к исходному состоянию списка.


  • Фильтрация списка осуществлется по параметрам: "Имя", "Возраст", "Администратор?".
    Для фильтрации по требуемому слову/положению требуется ввести текст в соответствующую форму под названием параметра.


  • Количество выводимых строк на одной странице определяется с помощью соответствующего инструмента на Панели администратора.
